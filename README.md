# Научный калькулятор (Scientific Calculator)

Полноценный калькулятор с графическим интерфейсом, написанный на чистом Python (Tkinter). Поддерживает базовую арифметику, тригонометрию, логарифмы, факториалы и многое другое.

## Как запустить

1. Убедитесь, что у вас установлен **Python 3.x** (проверить можно командой `python --version`).
2. Скачайте файл `calculator.py` из этого репозитория.
3. Запустите его через терминал или двойным кликом:
   
   ```bash
   python calculator.py

##  Функционал
Базовые операции: Сложение, вычитание, умножение, деление, возведение в степень (^), квадрат (x²), куб (x³).

Корни: Квадратный корень (√), кубический корень (∛).

Тригонометрия: Синус, косинус, тангенс (в градусах).

Логарифмы: Натуральный (ln) и десятичный (log).

Комбинаторика: Факториал (x!), субфакториал (!x).

Константы: Числа π и e.

Работа со знаками: Смена знака (±), десятичная точка.

Сброс: Полная очистка (C) и удаление одного символа (⌫).

Интерфейс: Почти современная темная тема.

##  Особенности реализации
Весь интерфейс построен на сетках (grid).

Универсальный обработчик унарных операций (позволяет применять функции к числу на экране).

Автоматическое сохранение ответа для продолжения вычислений.

##  Системные требования
Python 3.6 или новее.

ОС: Windows / macOS / Linux (везде, где работает Tkinter).

Никаких внешних библиотек не требуется!
